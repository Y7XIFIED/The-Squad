## ⚙️ Installation

### Prerequisites
- Node.js (v18.0.0 or higher recommended)
- npm (v9.0.0 or higher)

### Setup Instructions
1. Clone the repository:
```bash
git clone https://github.com/Y7XIFIED/The-Squad.git
cd The-Squad
```
2. Install the package dependencies:
```bash
npm install
```
3. Start the Vite development server:
```bash
npm run dev
```
